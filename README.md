# Product Quality & Field Testing Lab
**Lead QA Analyst:** Elena Butova  
**Focus:** Mobile Navigation, GIS Web Portals, and UX/Functional Testing

## Overview
This laboratory serves as a repository for deep-dive quality assurance audits. I leverage my daily 50-mile professional delivery routes as a "live stress-test environment" to identify edge cases, functional bugs, and UX friction points in high-congested transit corridors.

---

## 🔬 Current Test Suites

### 1. ArcGIS Navigator (Mobile GIS)
* **Methodology:** Real-world field testing during professional delivery operations.
* **Key Findings:** Latency in re-routing logic, UI glitches during high-speed maneuvers, and voice-command inconsistencies.
* [View Navigator Bug Reports](./Mobile-Navigation/ArcGIS-Navigator/)

### 2. Esri Web Ecosystem (Web Portals)
* **Methodology:** Exploratory testing during MOOC coursework and career portal navigation.
* **Key Findings:** Search filter persistence issues and accessibility/UX roadblocks.
* [View Web Portal Bug Reports](./Web-Portals/Esri-Web/)
